Automate your Pinterest affiliate strategy

Amazon Affiliate Manager is a personal automation tool that generates, reviews and publishes Pinterest pins for Amazon affiliate products — using AI-driven product scoring to maximise conversions.

Built and operated by JHM Freitas · Not a public product

How it works

End-to-end Pinterest automation

1
Product scoring
Weekly data analysis ranks Amazon products by sales rank, search trends and commission rate to identify the best products to promote.
2
AI pin generation
For each top-ranked product, AI generates keyword-rich pin titles, descriptions and hashtags optimised for Pinterest SEO and conversion.
3
Human review and approval
Each generated pin is reviewed and approved via a private dashboard before it is published — ensuring quality and compliance.
4
Scheduled Pinterest publishing
Approved pins are published to Pinterest automatically on a daily schedule using the Pinterest API, with affiliate links embedded in each pin.

Pinterest API usage

How this tool uses the Pinterest API

Operator
JHM Freitas — personal use only
Pinterest account
Single personal account owned by the operator
API operations used
POST /v5/pins — create pins on the operator's own boards
Posting frequency
1 pin per day, scheduled automatically
Data accessed
No Pinterest user data is read or stored. Only pins are created.
Third-party access
No third parties have access to this tool or the API credentials
