Privacy Policy

Last updated: April 2026

Overview

Affiliate Pin Manager is a private, personal automation tool used to generate, review and publish Pinterest pins for Amazon affiliate marketing purposes. This tool is not a public product and is not available to the general public.

Data we collect

This tool does not collect any personal data from third parties or end users. The only data processed is:

Pinterest content — pin titles, descriptions and hashtags generated by AI for the purpose of publishing to Pinterest. This content is created by the tool owner and published publicly on Pinterest.

Images — lifestyle photographs sourced from Pexels.com under their free commercial licence. Images are stored temporarily in a private cloud storage bucket and used solely for Pinterest publishing.

Authentication data — when accessing the review interface, a Google account sign-in is required. Only pre-approved email addresses can access the tool. No authentication data is shared with third parties.

Data storage

Pin content and images are stored in a private Supabase database and storage bucket. Access is restricted to authorised accounts only. No data is sold, shared or transferred to any third party.

Third-party services

This tool integrates with the following services:

Pinterest API — used to publish pins to a personal Pinterest account. Subject to Pinterest's Privacy Policy.

Google OAuth — used for authentication only. Subject to Google's Privacy Policy.

Pexels — used to source royalty-free images. Subject to Pexels' Privacy Policy.

Supabase — used for database and file storage. Subject to Supabase's Privacy Policy.

Google Gemini API — used to generate pin content. Subject to Google's Privacy Policy.

Cookies

This tool uses only functional session cookies required for authentication. No tracking or advertising cookies are used.

Your rights

As this is a private tool with no public user base, no personal data belonging to third parties is held. If you have any questions, contact the tool owner at business.jhmf@gmail.com.

Changes to this policy

This policy may be updated at any time. The last updated date at the top of this page reflects the most recent version.
